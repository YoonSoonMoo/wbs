Features

WBS 관리, 스트레스 없이

WBS를 작성하고 꾸준히 유지하는 일도 결국 공수입니다. Easy WBS는 코드 자동 생성, 날짜 자동 변환, AI 자연어 편집, 원클릭 통계·알림까지 반복 작업을 자동화해 관리자와 담당자 모두가 WBS에 쓰는 에너지를 최소로 줄여줍니다.

▦

계층형 WBS 그리드

Excel과 같은 인라인 편집, 복수 행 선택, 드래그 이동, Excel 붙여넣기를 지원합니다. 재귀 CTE 기반 트리 조회로 대규모 작업도 즉시 표시됩니다.

  • WBS 코드 자동 생성 (1.0 / 1.1 / 1.1.1)
  • 날짜 자유 입력 자동 변환 (2026/4/12 → 2026-04-12)
  • 컨텍스트 메뉴로 행 삽입·복제·삭제
◎

Gantt 차트 시각화

Frappe Gantt 기반의 인터랙티브 차트로 일정을 한눈에. 주간/월간/분기 단위 보기와 한국어 로케일을 지원합니다.

  • 계획 vs 실제 일정 비교
  • 마일스톤 하이라이트
  • 담당자별 색상 구분
✦

AI 어시스턴트

Claude 기반 자연어 명령으로 조회·추가·수정·삭제를 한 줄에. "지연된 태스크 보여줘"와 같은 질의를 그리드에 즉시 반영합니다.

  • 자연어 → WBS CRUD 자동 매핑
  • 일정 지연 분석 리포트
  • 결과를 그리드 필터로 즉시 적용
◴

주간 진척 통계

이번 주 진행분과 과거 4주 추이를 한 번에. KPI 카드·주차별 표·담당자별 분포·라인 차트로 진척률을 정량화합니다.

  • 전주 대비 delta 뱃지
  • 담당자별 완료/전체 비율 차트
  • 누적 진척률 추이 라인 차트
⇅

Excel · CSV 연동

기존에 사용하던 Excel 시트도 그대로. 한 번의 업로드로 전체 WBS를 가져오고, 필요할 때 바로 내보낼 수 있습니다.

  • Excel (.xlsx) / CSV (UTF-8 BOM) 지원
  • 탭 구분 텍스트 붙여넣기
  • 셀 다중 붙여넣기 자동 감지
✉

지연 알림 메일

계획 완료일을 넘긴 태스크를 담당자별로 그룹화해 SMTP HTML 메일로 자동 발송. 수동 독촉은 이제 그만.

  • 담당자별 지연 목록 자동 구성
  • 한 번의 클릭으로 일괄 발송
  • HTML 포맷으로 가독성 확보
Workflow

3단계로 끝내는 WBS 관리

복잡한 설정이나 학습 곡선 없이, 로그인 후 바로 사용하세요.

01

프로젝트 생성 및 멤버 할당

관리자가 프로젝트를 만들고 멤버를 개발자 / 뷰어 권한으로 초대합니다. 프로젝트별 접근 제어가 기본 적용됩니다.

02

WBS 작성 및 공수 배분

Excel 시트를 붙여넣거나 그리드에 직접 입력. WBS 코드·일정·담당자·공수를 실시간으로 저장합니다.

03

추적 · 통계 · 알림

Gantt 차트로 일정을 모니터링하고, 주간 통계로 진척률을 측정하며, 지연 시 자동 메일 알림으로 대응합니다.

권한 체계

Admin

전역 관리자. 프로젝트·유저 생성·수정·삭제, 권한 부여, 패스워드 리셋, 모든 프로젝트 접근 가능.

Developer

프로젝트 참여자. 할당된 프로젝트의 WBS 항목을 추가·수정·삭제할 수 있습니다.

Viewer

읽기 전용. 데이터 조회는 가능하나 편집 버튼은 자동으로 비활성화됩니다.
